Manatee Mentor User Manual

This guide explains how Mentors use Manatee to maintain profile details, manage requests, and work with mentorship goal records.

1. Account and Login

1. Register as Mentor or Both.

2. You are logged in immediately after account creation.

3. Return users sign in through the login page.

4. If 2FA is enabled, enter your authenticator app code to complete login.

2. Mentor Dashboard and Profile Fields

Your mentor dashboard contains:

1. Available to Mentor toggle at the top.

2. Functions you can Mentor (multi-select).

3. Your Location selector.

Use Update Profile to save changes.

3. Incoming Mentee Requests

Pending requests appear in Incoming Mentee Requests.

For each request you can:

1. Accept.

2. Decline.

3. Add a personal message.

Mentee receives your outcome by email.

4. Mentorships and Goals

The Mentorships section shows status history and actions.

Use the Goals button to open the dedicated mentorship goals page.

On the goals page you can:

1. Review the Next step record log (latest entries at the top).

2. Add new mentor-side entries.

3. View entries from both mentor and mentee.

In the dashboard list, a small unread count badge on Goals indicates new entries from the other party.

For accepted mentorships, use End with a reason when relationship completes or must close.

5. Emails You Receive

You may receive emails for:

1. New mentorship requests.

2. Reminder/resend requests.

3. Mentee cancellation of pending request.

4. Relationship ended notification from mentee.

5. New next-step goal entry notifications from your mentee.

6. Profile Page Settings

Use Profile to update email, password, and personal details.

You can also set up or disable 2FA using an authenticator app such as Authy.

7. Good Mentor Workflow

1. Keep availability accurate.

2. Keep function expertise current.

3. Respond to requests promptly.

4. Use clear messages when accepting or declining.

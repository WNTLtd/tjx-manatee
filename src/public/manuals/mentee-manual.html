Manatee Mentee User Manual

This guide explains how Mentees create requests, track statuses, and manage mentorship goals in Manatee.

1. Account and Login

1. Register as Mentee or Both.

2. You are logged in immediately after account creation.

3. Return users sign in from the login page.

4. If 2FA is enabled, enter your authenticator app code to complete login.

2. Profile Setup

Complete your profile details:

1. Title, First name, Surname, Job title.

2. Location (required for matching).

Save profile to activate mentor matching.

3. Finding Mentors

In Find Mentors:

1. Select a function.

2. Review available mentors for your location and function.

3. Click Request Mentorship.

4. Managing Pending Requests

In Your Mentorships, pending rows support:

1. Resend Request (send reminder to mentor).

2. Remove Request (cancel your pending request).

When you cancel, the mentor is notified by email.

5. Active Mentorships and Goals

When accepted, status changes to accepted.

Use the Goals button to open the dedicated mentorship goals page.

On the goals page you can:

1. Review the Next step record log (latest entries at the top).

2. Add new mentee-side entries.

3. View entries from both mentee and mentor.

In the dashboard list, a small unread count badge on Goals indicates new entries from the other party.

You can end active mentorships with an end reason from the list.

6. Emails You Receive

You may receive emails for:

1. Profile welcome/update confirmations.

2. Mentor accept/decline outcomes (with optional message).

3. Relationship-ended notifications.

4. New next-step goal entry notifications from your mentor.

7. Profile Page Settings

Use Profile to change email, password, and personal details at any time.

You can also set up or disable 2FA using an authenticator app such as Authy.

8. Good Mentee Workflow

1. Keep location updated.

2. Avoid duplicate requests for the same mentor/function.

3. Use resend only when needed.

4. End relationships with clear reason selection.
